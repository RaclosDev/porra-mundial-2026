# 🏆 Porra Mundial 2026

Una aplicación web moderna y en tiempo real para gestionar pronósticos y la clasificación de un grupo de amigos durante la Copa del Mundo de la FIFA 2026.

## ✨ Características Destacadas

- **📡 100% En Directo**: Conexión a la API oficial del Mundial. Los resultados, el calendario y los partidos en curso se actualizan automáticamente cada 30 segundos sin recargar la página. ¡Incluso suena una alerta de gol!
- **📊 Clasificación Automática**: Ranking general y tablas de grupos generadas al instante. El sistema maneja desempates complejos (diferencia de goles, enfrentamiento directo) y excluye a los equipos que aún no han debutado para mantener el orden real.
- **📺 Reproductor Integrado**: Haz una *Watch Party* directamente en la web. Incluye bypass de seguridad CORS mediante Cloudflare Workers para ver los partidos a pantalla completa nativa sin salir de la página.
- **📱 Social y Responsive**: Diseño premium *Glassmorphism* adaptado a móviles. Incluye un botón para compartir una captura instantánea de la clasificación por WhatsApp en un solo clic.

## 🚀 Tecnologías Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript ES6.
- **Backend/Datos**: Firebase Realtime Database.
- **Red/Streaming**: Cloudflare Workers proxy.
